# 🏠 Nestora

A modern and responsive property rental website built with React and Tailwind CSS.

## 📖 Overview

Nestora is a clean and modern real estate rental platform where users can browse available properties, view rental information, and explore property details through an attractive user interface.

The project focuses on creating a professional user experience, responsive layouts, reusable React components, and modern UI design principles.

---

## ✨ Features

* Modern and responsive design
* Property listing section
* Beautiful property cards
* Availability status indicators
* Property details overview
* Dynamic office status (Open / Closed)
* Reusable React components
* Mobile-first approach
* Clean and maintainable code structure

---

## 🛠️ Technologies

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/nilou1998/nestora.git
```

Navigate to the project:

```bash
cd nestora
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
